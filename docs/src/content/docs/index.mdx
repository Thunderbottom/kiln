## Features

<CardGrid>
<Card title="Simple UX" icon="rocket">
Seamlessly decrypt and expose secrets as environment variables to your apps with a single command, or export to a file.
</Card>

<Card title="Truly decentralized" icon="star">
No servers, databases, or external dependencies. Your secrets stay with your code.
</Card>

<Card title="Offline-first" icon="laptop">
Works completely offline with local files. Deploy anywhere, anytime.
</Card>

<Card title="Git-native" icon="github">
Encrypted files commit safely alongside your code. No separate infrastructure.
</Card>

<Card title="Role-based access" icon="random">
Built-in access control and recipient management. Add team members, create groups, and manage permissions effortlessly.
</Card>

<Card title="Multi-Environment" icon="approve-check">
Separate encrypted files for dev, staging, and production with independent access controls.
</Card>
</CardGrid>
