## Configuration Management

<CardGrid>
  <Card title="koanf Provider" icon="star">
    **Go Configuration Library**

    Load encrypted environment variables directly into [koanf](https://github.com/knadh/koanf) configuration with full transformation support.

    ```go
    import "github.com/knadh/koanf/providers/kiln"

    k.Load(kiln.Provider("kiln.toml", "", "production", kiln.Opt{
        Prefix: "API_",
        TransformFunc: transformKeys,
    }), nil)
    ```
