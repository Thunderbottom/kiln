kiln is distributed as a single binary with no external dependencies. Choose your preferred installation method below.

## Binary Releases

Download pre-built binaries for your platform from the [releases page](https://github.com/thunderbottom/kiln/releases).

<Tabs>
<TabItem label="Linux">

```bash
# Download the latest release, replacing the URL with the latest version from:
# https://github.com/Thunderbottom/kiln/releases/latest
curl -L https://github.com/Thunderbottom/kiln/releases/download/v1.0.0/kiln_1.0.0_linux_amd64.tar.gz

tar xzvf kiln_1.0.0_linux_amd64.tar.gz

# Make it executable and install
chmod +x kiln
sudo mv kiln /usr/local/bin/

# Verify installation
kiln --version
```

For ARM64 systems, use `kiln-linux-arm64`.

</TabItem>
<TabItem label="macOS">

```bash
# Download the latest release, replacing the URL with the latest version from:
# https://github.com/Thunderbottom/kiln/releases/latest

# Intel Macs
curl -L https://github.com/Thunderbottom/kiln/releases/download/v1.0.0/kiln_1.0.0_darwin_amd64.tar.gz

tar xzvf kiln_1.0.0_darwin_amd64.tar.gz

# Apple Silicon Macs (M1/M2)
curl -L https://github.com/Thunderbottom/kiln/releases/download/v1.0.0/kiln_1.0.0_darwin_arm64.tar.gz

tar xzvf kiln_1.0.0_darwin_arm64.tar.gz

# Make it executable and install
chmod +x kiln
sudo mv kiln /usr/local/bin/

# Verify installation
kiln --version
```

</TabItem>
<TabItem label="Windows">

1. Download `kiln-windows-amd64.exe` from the [releases page](https://github.com/thunderbottom/kiln/releases/latest)
2. Rename it to `kiln.exe`
3. Place it in a directory that's in your PATH
4. Open a new command prompt and run `kiln --version`

</TabItem>
</Tabs>

## Install from Source

If you have Go 1.23 or later installed:

```bash
go install github.com/thunderbottom/kiln@latest
```

This installs the latest release to your `$GOPATH/bin` directory.

## Development Build

To build from the development branch:

```bash
git clone https://github.com/thunderbottom/kiln.git
cd kiln
go mod download
make build

# Binary will be available at ./bin/kiln
```

## Verification

After installation, verify kiln is working:

```bash
# Check version
kiln --version

# View help
kiln --help

# Test key generation
kiln init key --help
```
